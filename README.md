# Бібліотека ігор з дошкою

Бібліотека ігор з дошкою, реалізована на C# без використання готових рушіїв. Проект включає повну систему логування, обробку помилок та сучасну архітектуру.

## Структура проекту

```
BoardGamesLibrary/
├── BoardGamesLibrary.Core/          # Базові класи та інтерфейси
├── BoardGamesLibrary.Logging/       # Система логування
├── BoardGamesLibrary.Games/         # Реалізації ігор
│   ├── Chess/                       # Шахи
│   ├── Checkers/                    # Шашки
│   ├── Reversi/                     # Reversi/Othello
│   └── ConnectFour/                 # Connect Four
├── BoardGamesLibrary.UI/            # Консольний інтерфейс
└── BoardGamesLibrary.Desktop/       # WPF десктопний інтерфейс
```

## Реалізовані ігри

### 1. Шахи (Chess)
- Повна реалізація правил шахів
- Підтримка всіх типів фігур
- Перевірка валідності ходів
- Визначення переможця

### 2. Шашки (Checkers)
- Класичні правила шашок
- Підтримка перетворення в дамку
- Обов'язкові стрибки
- Визначення переможця

### 3. Reversi/Othello
- Повна реалізація правил Reversi
- Автоматичне перевертання фігур
- Пропуск ходу, якщо немає валідних ходів
- Підрахунок фігур для визначення переможця

### 4. Connect Four
- Класична гра Connect Four
- Перевірка на 4 в ряд (горизонтально, вертикально, діагонально)
- Автоматичне розміщення фігур в найнижчу позицію

## Система логування

Проект включає повноцінну систему логування з підтримкою:
- **ConsoleLogger** - вивід в консоль з кольоровим форматуванням
- **FileLogger** - запис в файл з ротацією логів
- **CompositeLogger** - комбінація кількох логгерів
- Рівні логування: Debug, Info, Warning, Error, Critical

## Обробка помилок

Всі критичні операції обгорнуті в `ErrorHandler`, який:
- Логує всі помилки
- Забезпечує безпечне виконання операцій
- Дозволяє вказати значення за замовчуванням при помилці

## Архітектура

Проект використовує:
- **Інтерфейси** для абстракції (IBoard, IGame, IPiece, IMove)
- **Базові класи** для спільної логіки (BaseBoard, BaseMove)
- **Патерн Strategy** для різних ігор
- **Розділення відповідальності** між модулями

## Десктопна версія

Проект включає повноцінну WPF десктопну версію з:

- **Інтерактивними дошками** - керування мишкою
- **Візуальними ефектами** - анімації та підсвітка валідних ходів

### Особливості десктопної версії:

- Підсвітка валідних ходів

- Адаптивний дизайн з сучасними кольорами

## Використання

1. Запустіть програму
2. Оберіть гру з меню
3. Дотримуйтесь інструкцій на екрані для введення ходів
4. Гра автоматично визначає переможця або нічию

## Формати введення ходів

### Шахи та Шашки
```
рядок стовпець -> рядок стовпець
Наприклад: 1 0 -> 3 0
```

### Reversi
```
рядок стовпець
Наприклад: 2 3
```

### Connect Four
```
номер стовпця (0-6)
Наприклад: 3
```

## Логи

Логи зберігаються в папці `logs/` з ім'ям файлу `YYYY-MM-DD_game.log`.
