# Бібліотека ігор з дошкою

Бібліотека настільних ігор на C# (.NET 8) без використання готових ігрових рушіїв.
Проєкт містить ядро логіки, систему логування, консольний інтерфейс, WPF-додаток і набір юніт-тестів.

## Структура проєкту

```text
BoardGamesLibrary/
├── BoardGamesLibrary.Core/          # Базові інтерфейси та моделі
├── BoardGamesLibrary.Logging/       # Логування та ErrorHandler
├── BoardGamesLibrary.Games/         # Логіка ігор
│   ├── Chess/
│   ├── Checkers/
│   ├── Reversi/
│   └── ConnectFour/
├── BoardGamesLibrary.UI/            # Консольний застосунок
├── BoardGamesLibrary.Desktop/       # WPF застосунок
└── BoardGamesLibrary.Tests/         # Юніт-тести (xUnit)
```

## Реалізовані ігри

### Шахи
- Валідація ходів для всіх типів фігур
- Перевірка шаху/мату/пату
- Підтримка рокірування
- Підтримка взяття на проході (en passant)
- Перетворення пішака

### Шашки
- Класичні правила руху
- Обов'язкове взяття
- Підтримка дамки
- Підтримка множинних стрибків

### Reversi (Othello)
- Перевертання фішок у всіх напрямках
- Пропуск ходу, якщо немає валідних ходів
- Завершення гри при відсутності ходів у обох гравців

### Connect Four
- Розміщення фішки в найнижчу доступну клітинку стовпця
- Перевірка перемоги по горизонталі, вертикалі та діагоналях
- Нічия при заповненні поля

## Режими гри

Для WPF-версії доступні режими:
- **Два гравці**
- **Проти комп'ютера (AI)** з рівнем складності:
  - Легка
  - Середня
  - Важка

## Система логування

Підтримується:
- `ConsoleLogger`
- `FileLogger` (з ротацією по даті)
- `CompositeLogger`
- Рівні: `Debug`, `Info`, `Warning`, `Error`, `Critical`

Критичні операції виконуються через `ErrorHandler`.

## Архітектура

Ключові принципи:
- Абстракція через інтерфейси (`IGame`, `IBoard`, `IMove`, `IPiece`, `IAIPlayer`)
- Базові класи (`BaseBoard`, `BaseMove`, `BaseAIPlayer`)
- Розділення відповідальностей між модулями
- Клонування стану гри (`IGame.Clone`) для симуляцій AI без рефлексії

## Вимоги

- .NET 8 SDK
- Windows 10/11 (для WPF-версії)
- Visual Studio 2022+ або інший редактор з підтримкою .NET

## Збірка

```bash
dotnet build BoardGamesLibrary.sln
```

## Запуск

### Консольна версія

```bash
dotnet run --project BoardGamesLibrary.UI/BoardGamesLibrary.UI.csproj
```

### Десктопна версія (WPF)

```bash
dotnet run --project BoardGamesLibrary.Desktop/BoardGamesLibrary.Desktop.csproj
```

## Тести

Запуск усіх тестів:

```bash
dotnet test BoardGamesLibrary.Tests/BoardGamesLibrary.Tests.csproj
```

## Використання (WPF)

1. Запустіть застосунок.
2. Оберіть гру.
3. Виберіть режим: 2 гравці або проти AI.
4. Почніть гру.
5. Використовуйте кнопку **Нова гра** для швидкого перезапуску.

## Формати вводу (консоль)

### Шахи / Шашки

```text
рядок стовпець -> рядок стовпець
приклад: 1 0 -> 3 0
```

### Reversi

```text
рядок стовпець
приклад: 2 3
```

### Connect Four

```text
номер стовпця (0-6)
приклад: 3
```

## Логи
